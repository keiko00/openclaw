# CodeMaker 新人教学流程图

🎯 **CodeMaker 功能模块的新人教学流程指南**

## 📖 在线预览

### 方式一：GitHub Pages（推荐）
访问：`https://你的用户名.github.io/wuzucatProject/`

### 方式二：直接打开 HTML
下载仓库后，双击打开 `index.html` 即可在浏览器中查看。

## 📁 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 单文件版本，包含所有图片（Base64 嵌入），**推荐分享此文件** |
| `新人教学流程图.pdf` | PDF 版本，方便打印和离线查看 |
| `新人教学流程图.md` | Markdown 源码 |
| `assets/` | 图片资源文件夹（如需要单独使用图片） |

## 🚀 部署到 GitHub Pages

### 自动部署（推荐）

```bash
# 1. 初始化仓库并提交
git init
git add .
git commit -m "Initial commit: CodeMaker 新人教学流程图"

# 2. 创建 GitHub 仓库（在 GitHub 官网创建后）
git remote add origin https://github.com/你的用户名/wuzucatProject.git
git branch -M main
git push -u origin main

# 3. 启用 GitHub Pages
# 访问：https://github.com/你的用户名/wuzucatProject/settings/pages
# Source 选择 "main branch"，文件夹选择 "/"
# 点击 Save 即可
```

### 访问地址

启用 GitHub Pages 后，访问：
```
https://你的用户名.github.io/wuzucatProject/index.html
```
